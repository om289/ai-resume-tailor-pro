# Privacy And Safety Notes

Resumes can contain personal information such as names, emails, phone numbers, education history, and employment history.

For a demo project:

- Use sample resumes or remove personal contact details.
- Do not commit real resumes to GitHub.
- Do not commit API keys.
- Treat saved run history as private data.

For real deployment:

- Add authentication.
- Add a data deletion option.
- Encrypt backups.
- Keep audit logs.
- Tell users when text is sent to an external LLM API.
- Avoid inventing candidate experience; the prompt and UI both emphasize truthful edits.
